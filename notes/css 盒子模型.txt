

1.网页布局三大核心
    盒子模型、浮动、定位

2.网页布局过程
    ①准备好相关的网页元素，网页元素基本都是盒子box
    ②利用CSS设置好盒子样式，然后摆放到相应位置
    ③往盒子里装内容


3.盒子模型的组成

    ①border     边框

        border-width    边框粗细(px)
        border-style    边框样式:none（无/默认）、solid（实线）、dashed（虚线）、dotted（点线）
        border-color    边框颜色
        简写：
            border: 1px solid black(三者没有顺序要求)
    
        实际上能够单独设置上下左右的边框：
        border-top: 1px solid black
        border-bottom..
        ..

        边框实际会影响盒子的大小（边框实际是在盒子外围加的）
        
    ②content    内容

    ③padding    内边距

        padding-top: 像素值;
        ..
        简写：
            padding: 1个值（上下左右）
                     2个值（上下/左右）
                     3个值（上/左右/下）
                     4个值（上//右下/左，顺时针）
        
        padding也会影响盒子的大小
        但如果没有设定盒子的width/height属性，则不会撑大

    ④margin     外边距

        margin-top
        ..
        外边距的应用：
            可以让块级盒子水平居中，但盒子必须要指定了宽度以及左右边距都设置为auto
            如果是行内元素或行内块元素居中，则给父级设置即可

        父子盒子嵌套的问题（子盒子会贴着父盒子，哪怕单独设置外边距，也只会让整体一起移动）：
            给父元素定义上边框
            给父元素定义上内边距
            给父元素添加 overflow:hidden;
        
4.清除浏览器自带的内外边距
    * {
        padding: 0;
        margin: 0;
    }
    * {} 通配符选择器

    注：行内元素为了照顾兼容性，尽量只设置左右，不设置上下
        因为行内元素的宽高同内容有关

5.ps基本操作
    ①打开标尺   （ctrl + R）或（视图->标尺）
    ②右击标尺设置为像素
    ③利用矩形选框（会显示宽高的像素）

6.圆角边框
    border-radius       用于设置圆角
    border-radius: length(半径 px/%);

    两个数值按对角线设置


7.盒子阴影
    box-shadow: h-shadow v-shadow blur spread color inset;
    h-shadow    水平阴影*
    v-shadow    垂直阴影*
    blur        模糊距离
    spread      阴影尺寸
    color       阴影颜色
    inset       将外部阴影变为内部阴影

    鼠标经过则添加阴影效果：
    div:horver

8.文字阴影
    text-shadow: h-shadow v-shadow blur color;
