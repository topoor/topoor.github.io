

1. 网站的组成
    网站应用充裕主要分为两大部分：客户端和服务器端

    客户端：在浏览器中运行的部分，就是用户看到并与之交互的界面程序，使用 HTML、 CSS、 JS 构建
    服务器端：在服务器中运行的部分，负责存储数据和处理应用逻辑

2. Node 网站服务器
    能够提供网站访问的机器，接受客户端请求，对其做出响应

3. IP 地址
    互联网中设备的唯一标识

4. 域名
    上网所使用的网址
    虽然地址栏中输入的是网址，但是最终还是会将域名转换为 ip 才能访问到指定的网站服务器

5. 端口
    用来区分服务器电脑中提供不同的服务

6. URL
    统一资源定位符，网址

    URL的组成：
        传输协议://服务器IP或域名:端口/资源所在标识
        http://https://www.elever.xyz/demo/计时器移动端.html

    http: 超文本传输协议，提供了一种发布和接受 HTML页面的方法
    端口：默认是 80

7. 开发过程中客户端和服务器端说明
    在开发阶段，客户端和服务器使用同一台电脑，即开发人员电脑。
    客户端： 浏览器
    服务器端： Node

    本机域名： localhost
    本地IP： 127.0.0.1

8. 创建 web 服务器
    
    // 引用系统模块
    const http = require('http');

    // 创建 Web 服务器
    const app = http.createServer();

    // 客户端发送请求
    app.on('request',(req, res) => {
        // 响应
        // 结束响应
        res.end();
    });

    // 监听 3000 端口
    app.listen(3000);
    console.log('服务器已启动，监听3000端口，请访问 localhost:3000');

9. http 协议
    超文本传输协议
    是客户端和服务器端请求和应答的标准

    报文：
        ① 请求报文
            
            请求方式：
                Get     请求数据
                Post    发送数据

        ② 响应报文
